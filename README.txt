Exercícios Práticos de Lógica com JavaScript:
Descrição

Este projeto foi desenvolvido como parte de um conjunto de atividades destinadas à prática e compreensão dos fundamentos da programação web. Ele apresenta duas ferramentas simples — uma calculadora e um gerador de tabuada — construídas com HTML, CSS e JavaScript puro, sem dependências externas, com o objetivo de demonstrar como conceitos básicos podem ganhar vida em aplicações funcionais.

A proposta central do projeto é permitir que estudantes observem o fluxo completo da construção de uma aplicação web: desde a estrutura estática criada em HTML, passando pela camada visual definida no CSS, até chegar à lógica executada pelo JavaScript, responsável por manipular dinamicamente os elementos do documento por meio do DOM (Document Object Model).

Mais do que apenas apresentar funcionalidades, o projeto busca oferecer um ambiente limpo e didático para explorar eventos, validações, manipulação de textos, leitura de valores de entrada e geração dinâmica de elementos visuais; conceitos fundamentais para qualquer desenvolvedor iniciando na área.
Cada arquivo cumpre uma função clara e separada, favorecendo a compreensão da importância da organização estrutural em projetos web.

Uso

A utilização do projeto foi pensada para ser direta, sem etapas complexas de instalação ou configuração:

Abra o arquivo index.html em qualquer navegador moderno. Ele funciona como porta de entrada, oferecendo ao usuário as opções disponíveis e direcionando cada funcionalidade para sua respectiva página.

Na calculadora, o usuário pode inserir valores e executar operações como soma, subtração, multiplicação e divisão. O JavaScript interpreta os dados fornecidos, realiza o cálculo solicitado e retorna o resultado diretamente na interface.

No gerador de tabuada, o usuário informa um número e, com base nessa entrada, o script gera automaticamente uma sequência completa de multiplicações, exibida em uma lista organizada. O conteúdo é atualizado dinamicamente, permitindo que novos cálculos sejam feitos a qualquer momento.

Toda a lógica está encapsulada nos arquivos .js, de forma que o HTML permanece focado na estrutura e o CSS na aparência. Isso torna o projeto mais fácil de estudar e modificar, especialmente para iniciantes.

Essas funcionalidades têm como intenção não apenas executar cálculos, mas ilustrar como pequenos blocos de código podem transformar páginas estáticas em experiências interativas. O usuário pode alterar o número, testar novos valores, observar o comportamento da página e, assim, compreender na prática como o DOM reage às manipulações feitas pelo script.

Contribuição

Este projeto foi desenhado com simplicidade justamente para encorajar modificações, melhorias e experimentações por parte de quem desejar estudá-lo ou expandi-lo.
Existem diversas formas de contribuir:

Aprimorando a calculadora:
Adicionando novas operações, implementando tratamento de erros mais rigoroso, inserindo botões adicionais ou redesenhando o layout visual.

Expandindo o sistema de tabuada:
Possibilitando tabuada inversa, limites configuráveis, múltiplos números simultâneos ou até mesmo transformando o recurso em um pequeno jogo educativo.

Melhorando o design das páginas:
Reestruturando o CSS, aplicando animações simples, adicionando responsividade ou reorganizando a interface para torná-la mais moderna.

Refatorando o código JavaScript:
Separando funções, adotando padrões mais claros, otimizando a lógica e tornando o script mais legível para quem vier estudar posteriormente.

Criando novas funcionalidades:
O projeto foi feito para ser uma base. Nada impede que novas páginas sejam adicionadas, como conversores de unidades, pequenos desafios matemáticos, testes de lógica, entre outros.

A ideia é que este repositório sirva como um “laboratório de aprendizado”: um espaço onde iniciantes possam errar, testar, refazer, evoluir e visualizar de forma concreta o impacto de cada alteração.
Qualquer pessoa é bem-vinda para modificar, expandir ou reimaginar o projeto conforme sua própria necessidade de estudo.
